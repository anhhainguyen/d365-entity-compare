Privacy Policy for D365 Entity Compare

Publisher: Hai Nguyen

Effective date: May 7, 2026

D365 Entity Compare is a personal project that helps users compare Microsoft Dynamics 365 Finance & Operations entity metadata and selected OData records between two environments.

Data Access

The extension accesses content from user-selected D365 Finance & Operations websites in order to load entity metadata, compare records, and generate comparison results inside the browser.

Data Collection

This extension does not collect, sell, or transmit personal data to the developer or third parties for advertising, analytics, or marketing purposes.

Local Storage

The extension stores user configuration data locally in the browser, such as saved environment profiles, selected environments, and extension preferences, so users do not need to re-enter them each time.

Authentication

The extension uses the user's existing authenticated D365 browser session to request data from the selected environment. Login credentials and passwords are not collected by the developer.

Data Sharing

No user data is sold or shared with third parties, except when required by law.

Purpose Limitation

Any accessed website content is used only to provide the extension's comparison functionality.
